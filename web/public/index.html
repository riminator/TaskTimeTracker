📊 Import from CSV/Excel

Upload a CSV or Excel file with your time entries. Supported columns: Date, Project/Client Name, Meeting/Project Title, Start Time, End Time, Duration (hrs), Notes, Employee(s) Attended Name(s)

📅 Import from Calendar (ICS)

Upload an ICS calendar file exported from Outlook, Google Calendar, or other calendar applications. Meetings will be automatically classified.

📖 Import Instructions

CSV/Excel Format

  • First row should contain column headers
  • Date format: YYYY-MM-DD or M/D/YY
  • Time format: HH:MM AM/PM or HH:MM:SS
  • Duration: decimal hours (e.g., 1.5 for 90 minutes)

Calendar (ICS) Files

  • Outlook: File → Save Calendar → iCalendar Format (.ics)
  • Google Calendar: Settings → Import & Export → Export
  • Apple Calendar: File → Export → Export

Auto-Classification

Imported entries are automatically classified based on meeting titles and organizers. You can review and edit classifications after import.
